<h2>Set up </h2>

1. Project need to set up all module from the begining:

```
npm run install:ALl
```

2. Run Front-End independently

```
npm run start:front_end

```

3. Run Back-End independently

```

npm run start:back_end

```

4. Run All Project:

```
npm run start:All
```

5. Finnish Build UI:

```
npm run build:All

```
